# Portfolio Website

This is a simple, responsive portfolio website built with React and Vite. It showcases personal projects, background information, and links to social profiles.

## ✨ Features

- 🗓️ **Dynamic age calculation** based on your birthdate.
- 💻 **Project list** with descriptions and links to GitHub repositories.
- 🌙 **Dark mode toggle**, stored via sessionStorage.
- 📅 **Footer displays the current year** automatically.
- 📱 **Fully responsive design** across devices.

## 🛠️ Technologies Used

- **Frontend Framework**: React (with Vite)
- **Languages**: HTML, CSS, JavaScript
- **Dark Mode**: Implemented manually using session storage

## 🚀 Deployment

The site can be deployed with Docker and served behind a reverse proxy like Traefik for automatic HTTPS and scalability.
